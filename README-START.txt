========================================
   FOODBRIDGE - HOW TO START
========================================

METHOD 1: Using Batch File (Easiest)
-------------------------------------
1. Right-click on "LAUNCH.bat"
2. Select "Run as administrator" (if needed)
3. Or right-click and select "Open with" -> "Command Prompt"

METHOD 2: Using Terminal
-------------------------------------
1. Open Command Prompt or PowerShell
2. Navigate to FoodBridge folder:
   cd "C:\Users\ASUS\OneDrive\Desktop\FoodBridge"
3. Run: npm start
4. Open browser and go to: http://localhost:4000/login.html

METHOD 3: Using START.html
-------------------------------------
1. Double-click "START.html" file
2. It will try to open the login page
3. Make sure server is running first (npm start)

METHOD 4: Direct Browser
-------------------------------------
1. First, open terminal and run: npm start
2. Then open browser
3. Type: http://localhost:4000/login.html

========================================
   QUICK LINKS (when server is running)
========================================
- Login: http://localhost:4000/login.html
- Donor Post: http://localhost:4000/donor_post.html
- Receiver View: http://localhost:4000/receiver_view.html
- Home: http://localhost:4000/

========================================
   TROUBLESHOOTING
========================================
If batch file doesn't work:
- Right-click -> Properties -> Unblock (if available)
- Try running from terminal instead
- Check if Node.js is installed: node --version
